A curated list of awesome Dash (plotly) resources

> [Dash](https://plot.ly/products/dash/) is a productive Python framework for building web applications.
> Written on top of Flask, Plotly.js, and React.js, Dash is ideal for building data visualization apps with highly custom user interfaces in pure Python. It's particularly suited for anyone who works with data in Python.

## App Examples

- [Uber Rides](https://dash.gallery/dash-uber-rides-demo/) - Displays all of the Uber rides in New York City in 2014. Pandas on the backend filters a 0.5gig datafile. [Source Code.](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-uber-rides-demo) [![GitHub stars](https://img.shields.io/github/stars/plotly/dash-sample-apps/tree/main/apps/dash-uber-rides-demo?style=flat)](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-uber-rides-demo/stargazers)
- [Drug Discovery](https://dash.gallery/dash-drug-discovery/) - Displays a description of the drug as you hover over points in the graph. - [Live Wind Streaming](https://dash.gallery/dash-wind-streaming/) - Continually queries a SQL database and displays live charts of wind speed and wind direction. [Source Code.](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-wind-streaming) [![GitHub stars](https://img.shields.io/github/stars/plotly/dash-sample-apps/tree/main/apps/dash-wind-streaming?style=flat)](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-wind-streaming/stargazers)
- [3D Yield Curve](https://dash.gallery/dash-yield-curve/) - Adapted from NYTimes's excellent [A 3-D View of a Chart That Predicts The Economic Future: The Yield Curve](https://www.nytimes.com/interactive/2015/03/19/upshot/3d-yield-curve-economic-growth.html). [Source Code.](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-yield-curve) [![GitHub stars](https://img.shields.io/github/stars/plotly/dash-sample-apps/tree/main/apps/dash-yield-curve?style=flat)](https://github.com/plotly/dash-sample-apps/tree/main/apps/dash-yield-curve/stargazers)
- [Visualize Earthquakes with Plotly Dash](https://dash-earthquakes-production-45eyyotfta-ey.a.run.app/) - Great UI and usage of geospatial analytics with Dash. Includes [basic unit tests](https://github.com/jackdbd/dash-earthquakes/tree/master/tests) [![GitHub stars](https://img.shields.io/github/stars/jackdbd/dash-earthquakes/tree/master/tests?style=flat)](https://github.com/jackdbd/dash-earthquakes/tree/master/tests/stargazers).
