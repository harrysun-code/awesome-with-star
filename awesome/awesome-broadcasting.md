## Animation, Graphics & Video Playout

* [Aurena](https://github.com/thaytan/aurena) [![GitHub stars](https://img.shields.io/github/stars/thaytan/aurena?style=flat)](https://github.com/thaytan/aurena/stargazers) - A network distributed media playback system.
* [Blender](https://projects.blender.org/blender/blender) - 3D creation suite supporting 3D modelling, animation, motion tracking, video editing, and more (overview [here](https://developer.blender.org/)).
* [Bridge](https://github.com/svt/bridge) [![GitHub stars](https://img.shields.io/github/stars/svt/bridge?style=flat)](https://github.com/svt/bridge/stargazers) - Next generation graphics control software, with extension support.
* [caspar-obs-client](https://github.com/michalramus/caspar-obs-client) [![GitHub stars](https://img.shields.io/github/stars/michalramus/caspar-obs-client?style=flat)](https://github.com/michalramus/caspar-obs-client/stargazers) - Python GUI application for seamless integration between CasparCG media servers and OBS Studio.
* [CasparCG](http://www.casparcg.com/) - A professional graphics and video play-out software, proven in 24/7 broadcasts since 2006.
* [ffplayout](https://github.com/ffplayout/ffplayout) [![GitHub stars](https://img.shields.io/github/stars/ffplayout/ffplayout?style=flat)](https://github.com/ffplayout/ffplayout/stargazers) - Rust and FFmpeg based playout from folder or playlists.
* [Macadam](https://github.com/Streampunk/macadam) [![GitHub stars](https://img.shields.io/github/stars/Streampunk/macadam?style=flat)](https://github.com/Streampunk/macadam/stargazers) - Blackmagic Node.js bindings that support HTML/CSS (via [Electron](https://www.electronjs.org/)) and SVG (via [Sevruga](https://github.com/Streampunk/sevruga) [![GitHub stars](https://img.shields.io/github/stars/Streampunk/sevruga?style=flat)](https://github.com/Streampunk/sevruga/stargazers)) graphics.
* [Nebula](https://github.com/nebulabroadcast) [![GitHub stars](https://img.shields.io/github/stars/nebulabroadcast?style=flat)](https://github.com/nebulabroadcast/stargazers) - Media asset management and broadcast automation system.
* [NodeCG](https://www.nodecg.dev/) - Broadcast graphics rendered in a browser using Node.js.
* [OGraf](https://github.com/ebu/ograf) [![GitHub stars](https://img.shields.io/github/stars/ebu/ograf?style=flat)](https://github.com/ebu/ograf/stargazers) - Open specification for HTML based Graphics, used in live television and post production workflows.
* [Open Playout Automation](https://github.com/jaskie/PlayoutAutomation) [![GitHub stars](https://img.shields.io/github/stars/jaskie/PlayoutAutomation?style=flat)](https://github.com/jaskie/PlayoutAutomation/stargazers) - A CasparCG-based MCR play-out system.
* [ossia](https://ossia.io/) - A free and open-source intermedia sequencer.
* [Sofie - TV Automation](https://github.com/Sofie-Automation/Sofie-TV-automation) [![GitHub stars](https://img.shields.io/github/stars/Sofie-Automation/Sofie-TV-automation?style=flat)](https://github.com/Sofie-Automation/Sofie-TV-automation/stargazers) - MOS-driven automation system for news casts, with many libraries for e.g. device control.
* [SPX - Graphics Controller](https://github.com/TuomoKu/SPX-GC) [![GitHub stars](https://img.shields.io/github/stars/TuomoKu/SPX-GC?style=flat)](https://github.com/TuomoKu/SPX-GC/stargazers) - Graphics control client for live video productions and live streams.
* [StreamShapers - Ferryman](https://github.com/Streamshapers/StreamShapers-Ferryman) [![GitHub stars](https://img.shields.io/github/stars/Streamshapers/StreamShapers-Ferryman?style=flat)](https://github.com/Streamshapers/StreamShapers-Ferryman/stargazers) - Web App to generate HTML-Graphics from Lottie.JSON files / from Adobe AfterEffects.
* [Studio TV Player](https://github.com/jaskie/StudioTVPlayer) [![GitHub stars](https://img.shields.io/github/stars/jaskie/StudioTVPlayer?style=flat)](https://github.com/jaskie/StudioTVPlayer/stargazers) - Simple TV studio player with SDI, NDI and MPEG TS outputs.

## Codecs

* [FLAC](https://www.xiph.org/flac/) - Free Lossless Audio Coding, used by some broadcasters for audio exchange, storage.
* [Lame](https://lame.sourceforge.io/) - A high quality MPEG Audio Layer III (MP3) encoder.
* [opencore-amr](https://sourceforge.net/projects/opencore-amr/) - Audio codecs extracted from Android Open Source Project, including AAC.
* [Opus](https://www.opus-codec.org/) - A totally open, royalty-free, highly versatile audio codec.
* [Turing Codec](https://github.com/bbc/turingcodec) [![GitHub stars](https://img.shields.io/github/stars/bbc/turingcodec?style=flat)](https://github.com/bbc/turingcodec/stargazers) - An H.265/HEVC open source software encoder designed for fast and efficient video compression.
* [TwoLame](https://www.twolame.org/) - An MPEG Audio Layer 2 (MP2) encoder.
