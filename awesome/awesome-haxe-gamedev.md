## Game engines

Those are Haxe 4 compatible game engines
* [Armory (Kha)](https://github.com/armory3d/armory) [![GitHub stars](https://img.shields.io/github/stars/armory3d/armory?style=flat)](https://github.com/armory3d/armory/stargazers) - An open-source 3D game engine with full Blender integration (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [Away3D](https://github.com/openfl/away3d) [![GitHub stars](https://img.shields.io/github/stars/openfl/away3d?style=flat)](https://github.com/openfl/away3d/stargazers) - An open source, real-time 3D engine for OpenFL (`Web`, `Mobile`, `Desktop`).
* [ceramic](https://github.com/ceramic-engine/ceramic) [![GitHub stars](https://img.shields.io/github/stars/ceramic-engine/ceramic?style=flat)](https://github.com/ceramic-engine/ceramic/stargazers) - Cross-platform 2D framework (`Web`, `Mobile`, `Desktop`, `Unity`).
* [HaxeFlixel (OpenFL)](https://github.com/HaxeFlixel/flixel) [![GitHub stars](https://img.shields.io/github/stars/HaxeFlixel/flixel?style=flat)](https://github.com/HaxeFlixel/flixel/stargazers) - Free, cross-platform 2D game engine powered by OpenFL (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [Haxegon (OpenFL)](https://github.com/haxegon/haxegon) [![GitHub stars](https://img.shields.io/github/stars/haxegon/haxegon?style=flat)](https://github.com/haxegon/haxegon/stargazers) - A programming library for beginners. Powered by OpenFL and Starling (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [Heaps](https://github.com/HeapsIO/heaps) [![GitHub stars](https://img.shields.io/github/stars/HeapsIO/heaps?style=flat)](https://github.com/HeapsIO/heaps/stargazers) - High Performance Game Framework (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [hxdefold](https://github.com/hxdefold/hxdefold) [![GitHub stars](https://img.shields.io/github/stars/hxdefold/hxdefold?style=flat)](https://github.com/hxdefold/hxdefold/stargazers) - Haxe/Lua externs for Defold game engine (`Web`, `Mobile`, `Desktop`).
* [OpenFL](https://github.com/openfl/openfl) [![GitHub stars](https://img.shields.io/github/stars/openfl/openfl?style=flat)](https://github.com/openfl/openfl/stargazers) - Interactive game and app development library (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [Starling](https://github.com/openfl/starling) [![GitHub stars](https://img.shields.io/github/stars/openfl/starling?style=flat)](https://github.com/openfl/starling/stargazers) - The "Cross-Platform Game Engine", a popular Stage3D framework (`Web`, `Mobile`, `Desktop`).
* [Stencyl (OpenFL)](https://github.com/Stencyl/stencyl-engine) [![GitHub stars](https://img.shields.io/github/stars/Stencyl/stencyl-engine?style=flat)](https://github.com/Stencyl/stencyl-engine/stargazers) - Create Flash, HTML5, iOS, Android, and desktop games with no code (`Mobile`, `Desktop`).
* [unreal.hx](https://github.com/proletariatgames/unreal.hx) [![GitHub stars](https://img.shields.io/github/stars/proletariatgames/unreal.hx?style=flat)](https://github.com/proletariatgames/unreal.hx/stargazers) - Haxe Integration for Unreal (`Web`, `Mobile`, `Desktop`, `Consoles`).
* [HxGodot (Godot 4.0)](https://github.com/HxGodot/hxgodot) [![GitHub stars](https://img.shields.io/github/stars/HxGodot/hxgodot?style=flat)](https://github.com/HxGodot/hxgodot/stargazers) - A Haxe GDExtension for Godot 4 (`Web`, `Mobile`, `Desktop`, `Consoles`).

## Architecture
```
IoC == Inversion of Control  
EC == Entity Component  
ECS == Entity-Component-System
FSM == Finite State Machine
MVC == Model View Controller
```

* [awe6](https://github.com/hypersurge/awe6) [![GitHub stars](https://img.shields.io/github/stars/hypersurge/awe6?style=flat)](https://github.com/hypersurge/awe6/stargazers) - `IoC`, `EC` - The inverted game framework, is a development tool focused on Future Proofing.
* [ecx](https://github.com/eliasku/ecx) [![GitHub stars](https://img.shields.io/github/stars/eliasku/ecx?style=flat)](https://github.com/eliasku/ecx/stargazers) - `ECS` - An Entity Component System framework.
* [hexMachina](https://github.com/DoclerLabs/hexCore) [![GitHub stars](https://img.shields.io/github/stars/DoclerLabs/hexCore?style=flat)](https://github.com/DoclerLabs/hexCore/stargazers) - `MVC` - A powerful multi-modular MVC framework.
* [OSIS](https://github.com/Dvergar/OSIS) [![GitHub stars](https://img.shields.io/github/stars/Dvergar/OSIS?style=flat)](https://github.com/Dvergar/OSIS/stargazers) - `ECS` - Entity Component System architecture with networking support.
