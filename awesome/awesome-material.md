# Awesome Material Design [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)

A curated list of Google's [material design](http://www.google.com/design/spec) based libraries, resources and awesome things.

Icons, Fonts and Colors
==
  - [material-design-icons](https://github.com/google/material-design-icons) [![GitHub stars](https://img.shields.io/github/stars/google/material-design-icons?style=flat)](https://github.com/google/material-design-icons/stargazers) — Material Design icons by Google.
  - [material-design-iconic-font](https://github.com/zavoloklom/material-design-iconic-font) [![GitHub stars](https://img.shields.io/github/stars/zavoloklom/material-design-iconic-font?style=flat)](https://github.com/zavoloklom/material-design-iconic-font/stargazers) — Material Design Iconic Font and CSS toolkit.
  - [material-design-fonticons](https://github.com/designjockey/material-design-fonticons) [![GitHub stars](https://img.shields.io/github/stars/designjockey/material-design-fonticons?style=flat)](https://github.com/designjockey/material-design-fonticons/stargazers) — Material Design Fonticons is a font converted version of the Google Material Design Icon set.
  - [material-colors](https://github.com/shuhei/material-colors) [![GitHub stars](https://img.shields.io/github/stars/shuhei/material-colors?style=flat)](https://github.com/shuhei/material-colors/stargazers) — Colors of Google's Material Design made available to coders.
  - [material-color](https://github.com/mrmlnc/material-color) [![GitHub stars](https://img.shields.io/github/stars/mrmlnc/material-color?style=flat)](https://github.com/mrmlnc/material-color/stargazers) — The colour palette, based on Google's Material Design, for use in your project.
  - [LESS-Material-design-colors](https://github.com/tisign/LESS-Material-design-colors) [![GitHub stars](https://img.shields.io/github/stars/tisign/LESS-Material-design-colors?style=flat)](https://github.com/tisign/LESS-Material-design-colors/stargazers) — A .less file with al the Google material design colors.
  - [sass-material-colors](https://github.com/minusfive/sass-material-colors) [![GitHub stars](https://img.shields.io/github/stars/minusfive/sass-material-colors?style=flat)](https://github.com/minusfive/sass-material-colors/stargazers) — An easy way to use Google's Material Design colors in your Sass/Scss project.
  - [Google-Material-Design-Font-Icon](https://github.com/Seb-L/Google-Material-Design-Font-Icon) [![GitHub stars](https://img.shields.io/github/stars/Seb-L/Google-Material-Design-Font-Icon?style=flat)](https://github.com/Seb-L/Google-Material-Design-Font-Icon/stargazers) — Font Icon based on Google Material Design Icon set.
  - [Material Design Icons](https://materialdesignicons.com/) — Community generated Material Design icon pack.
  - [Android Asset Studio](https://romannurik.github.io/AndroidAssetStudio/) — A web-based set of tools for generating graphics and other assets that would eventually be in an Android application's res/ directory.
