## Standalone Editors Built on Draft.js

* [Draft WYSIWYG](https://github.com/bkniffler/draft-wysiwyg) [![GitHub stars](https://img.shields.io/github/stars/bkniffler/draft-wysiwyg?style=flat)](https://github.com/bkniffler/draft-wysiwyg/stargazers) - WYSIWYG editor that with drag&drop, resizing & tooltips.

* [Megadraft](https://github.com/globocom/megadraft) [![GitHub stars](https://img.shields.io/github/stars/globocom/megadraft?style=flat)](https://github.com/globocom/megadraft/stargazers) - A rich text editor with a nice default base of plugins and extensibility.
* [Medium Draft](https://github.com/brijeshb42/medium-draft) [![GitHub stars](https://img.shields.io/github/stars/brijeshb42/medium-draft?style=flat)](https://github.com/brijeshb42/medium-draft/stargazers) - Medium-like rich text editor with a focus on keyboard shortcuts.
* [React-Draft-Wyiswyg](https://github.com/jpuri/react-draft-wysiwyg) [![GitHub stars](https://img.shields.io/github/stars/jpuri/react-draft-wysiwyg?style=flat)](https://github.com/jpuri/react-draft-wysiwyg/stargazers) - A WYISWYG editor, with various text editing options and corresponding HTML generation.
* [Dante 2](https://github.com/michelson/dante2) [![GitHub stars](https://img.shields.io/github/stars/michelson/dante2?style=flat)](https://github.com/michelson/dante2/stargazers) - Just another Medium clone built on top of DraftJs.
* [Last Draft](https://github.com/vacenz/last-draft) [![GitHub stars](https://img.shields.io/github/stars/vacenz/last-draft?style=flat)](https://github.com/vacenz/last-draft/stargazers) - A Draft editor built with Draft.js plugins.
* [Z-Editor](https://github.com/Z-Editor/Z-Editor) [![GitHub stars](https://img.shields.io/github/stars/Z-Editor/Z-Editor?style=flat)](https://github.com/Z-Editor/Z-Editor/stargazers) - Online Z-notations editor.
