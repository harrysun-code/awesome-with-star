## Tools

Fun
- [pwnagotchi](https://github.com/evilsocket/pwnagotchi) [![GitHub stars](https://img.shields.io/github/stars/evilsocket/pwnagotchi?style=flat)](https://github.com/evilsocket/pwnagotchi/stargazers) - Your AI pet that hacks WiFI to grow. It's super cute.

DDoS

Wi-Fi.
- [trackerjacker](https://github.com/calebmadrigal/trackerjacker) [![GitHub stars](https://img.shields.io/github/stars/calebmadrigal/trackerjacker?style=flat)](https://github.com/calebmadrigal/trackerjacker/stargazers) - Maps and tracks Wi-Fi networks and devices through raw 802.11 monitoring.
- [wifiphisher](https://github.com/wifiphisher/wifiphisher) [![GitHub stars](https://img.shields.io/github/stars/wifiphisher/wifiphisher?style=flat)](https://github.com/wifiphisher/wifiphisher/stargazers) - Create rogue access point.

Wireless
- [WHAD](https://github.com/whad-team/whad-client) [![GitHub stars](https://img.shields.io/github/stars/whad-team/whad-client?style=flat)](https://github.com/whad-team/whad-client/stargazers) - A powerful framework to perform various kinds of Wireless attacks.

IPv6
- [Chiron](https://github.com/aatlasis/Chiron) [![GitHub stars](https://img.shields.io/github/stars/aatlasis/Chiron?style=flat)](https://github.com/aatlasis/Chiron/stargazers) - An IPv6 security assessment framework.
- [mitm6](https://github.com/fox-it/mitm6) [![GitHub stars](https://img.shields.io/github/stars/fox-it/mitm6?style=flat)](https://github.com/fox-it/mitm6/stargazers) - Performs MiTM for IPv6.

Measurements
- [mtraceroute](https://github.com/rwhalb/mtraceroute) [![GitHub stars](https://img.shields.io/github/stars/rwhalb/mtraceroute?style=flat)](https://github.com/rwhalb/mtraceroute/stargazers) - Create cool graphs over multiple traceroute analysis.

Protocols
- [Cotopaxi](https://github.com/Samsung/cotopaxi) [![GitHub stars](https://img.shields.io/github/stars/Samsung/cotopaxi?style=flat)](https://github.com/Samsung/cotopaxi/stargazers) - Set of tools for security testing of Internet of Things devices using specific network IoT protocols (AMQP, CoAP, DTLS, HTCPCP, KNX, mDNS, MQTT, MQTT-SN, QUIC, RTSP, SSDP) .

- [routopsy](https://github.com/sensepost/routopsy) [![GitHub stars](https://img.shields.io/github/stars/sensepost/routopsy?style=flat)](https://github.com/sensepost/routopsy/stargazers) - Toolkit to attack DRP & FHRP.
- [TorPylle](https://github.com/cea-sec/TorPylle) [![GitHub stars](https://img.shields.io/github/stars/cea-sec/TorPylle?style=flat)](https://github.com/cea-sec/TorPylle/stargazers) - Implementation of the OR (TOR) protocol.

- [OpenBSD](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Fsearch%3Fq%3Dscapy%2Brepo%253Aopenbsd%252Fsrc%2Bpath%253Aregress%252F%26type%3DCode%26ref%3Dadvsearch%26l%3D%26l%3D) [![GitHub stars](https://img.shields.io/github/stars/login?return_to=https%3A%2F%2Fgithub.com%2Fsearch%3Fq%3Dscapy%2Brepo%253Aopenbsd%252Fsrc%2Bpath%253Aregress%252F%26type%3DCode%26ref%3Dadvsearch%26l%3D%26l%3D?style=flat)](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Fsearch%3Fq%3Dscapy%2Brepo%253Aopenbsd%252Fsrc%2Bpath%253Aregress%252F%26type%3DCode%26ref%3Dadvsearch%26l%3D%26l%3D/stargazers) - IPv6 stack testing suite.
