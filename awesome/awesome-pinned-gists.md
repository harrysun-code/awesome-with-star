## GitHub

Displaying more detailed GitHub user data in a pinned gist.

- [activity-box](https://github.com/JasonEtco/activity-box) [![GitHub stars](https://img.shields.io/github/stars/JasonEtco/activity-box?style=flat)](https://github.com/JasonEtco/activity-box/stargazers) - Update a pinned gist to contain the latest activity of a GitHub user.
  - [github-activity-box](https://github.com/maxchang3/github-activity-box) [![GitHub stars](https://img.shields.io/github/stars/maxchang3/github-activity-box?style=flat)](https://github.com/maxchang3/github-activity-box/stargazers) - A TypeScript rewrite of activity-box, published on npm for easy GitHub Actions integration, with improved type safety, customizability, and using the GraphQL API to retrieve issues and PRs authored by the specified user.
- [github-stats-box](https://github.com/bokub/github-stats-box) [![GitHub stars](https://img.shields.io/github/stars/bokub/github-stats-box?style=flat)](https://github.com/bokub/github-stats-box/stargazers) - Update a pinned gist to contain your GitHub statistics.
- [lang-box](https://github.com/inokawa/lang-box) [![GitHub stars](https://img.shields.io/github/stars/inokawa/lang-box?style=flat)](https://github.com/inokawa/lang-box/stargazers) - Update a pinned gist to contain languages of your recent commits in GitHub
  - [recent-languages-box](https://github.com/liby/recent-languages-box) [![GitHub stars](https://img.shields.io/github/stars/liby/recent-languages-box?style=flat)](https://github.com/liby/recent-languages-box/stargazers) - A TypeScript + Bun rewrite of lang-box that updates a pinned gist with your recent GitHub commit language statistics. It offers improved type safety, customizability, and uses a more scientific approach to sorting languages.
  - [github-lang-box](https://github.com/maxchang3/github-lang-box) [![GitHub stars](https://img.shields.io/github/stars/maxchang3/github-lang-box?style=flat)](https://github.com/maxchang3/github-lang-box/stargazers) - A TypeScript rewrite of lang-box, published on npm for easy GitHub Actions integration, with improved type safety, customizability, and private repository access.
- [productive-box](https://github.com/maxam2017/productive-box) [![GitHub stars](https://img.shields.io/github/stars/maxam2017/productive-box?style=flat)](https://github.com/maxam2017/productive-box/stargazers) - Update a pinned gist to contain your most productive hours during the day.
