> A curated list of awesome things related to the TALL stack.

The [TALL stack](https://tallstack.dev/) is a full stack development solution featuring some of the libraries built by the Laravel community. It features tools that allows you to write apps with a focus on backend and without having to write too much JavaScript, thus allowing you to focus on a smaller set of programming tools.

## Libraries

- [Alptail](https://www.alptail.com) - A collection of open-source UI components, using Tailwind CSS and Alpine.js.
- [Blade Icons](https://github.com/blade-ui-kit/blade-icons) [![GitHub stars](https://img.shields.io/github/stars/blade-ui-kit/blade-icons?style=flat)](https://github.com/blade-ui-kit/blade-icons/stargazers) - A package to easily make use of SVG icons in your Laravel Blade views.
- [Blade Script](https://github.com/cbl/blade-script) [![GitHub stars](https://img.shields.io/github/stars/cbl/blade-script?style=flat)](https://github.com/cbl/blade-script/stargazers) - A package to easily add transpiled & minified scripts to your Blade components.
- [Blade Style](https://github.com/cbl/blade-style) [![GitHub stars](https://img.shields.io/github/stars/cbl/blade-style?style=flat)](https://github.com/cbl/blade-style/stargazers) - A package to easily minify styles and make use of sass, less, etc. in your Blade components.
- [Blade UI Kit](https://blade-ui-kit.com/) - A set of renderless components to utilise in your Laravel Blade views.
- [Docdress](https://github.com/aw-studio/docdress) [![GitHub stars](https://img.shields.io/github/stars/aw-studio/docdress?style=flat)](https://github.com/aw-studio/docdress/stargazers) - A package to create markdown documentations from GitHub repositories in a Laravel-like design.
- [Filament](https://filamentadmin.com) - The elegant TALL stack admin panel for Laravel artisans.
- [Laravel Form Components](https://github.com/pascalbaljetmedia/laravel-form-components) [![GitHub stars](https://img.shields.io/github/stars/pascalbaljetmedia/laravel-form-components?style=flat)](https://github.com/pascalbaljetmedia/laravel-form-components/stargazers) - A set of Blade components to rapidly build forms with Tailwind CSS Custom Forms and Bootstrap 4.
- [Laravel TALL Preset](https://github.com/laravel-frontend-presets/tall) [![GitHub stars](https://img.shields.io/github/stars/laravel-frontend-presets/tall?style=flat)](https://github.com/laravel-frontend-presets/tall/stargazers) - A front-end preset for Laravel to scaffold an application using the TALL stack.
- [Laravel TALL Preset](https://github.com/use-preset/laravel-tall/) [![GitHub stars](https://img.shields.io/github/stars/use-preset/laravel-tall/?style=flat)](https://github.com/use-preset/laravel-tall//stargazers) - A preset for installing the TALL stack with just one command.
- [Laravel TTALL Preset](https://github.com/pktharindu/ttall) [![GitHub stars](https://img.shields.io/github/stars/pktharindu/ttall?style=flat)](https://github.com/pktharindu/ttall/stargazers) - A preset for Turbolinks + TALL stack, bundled with some helpful packages and their configurations.
- [Laravel Views](https://github.com/Gustavinho/laravel-views) [![GitHub stars](https://img.shields.io/github/stars/Gustavinho/laravel-views?style=flat)](https://github.com/Gustavinho/laravel-views/stargazers) - Laravel package to create beautiful common views like data tables using the TALL stack.
- [Livewire Datatables](https://github.com/mediconesystems/livewire-datatables) [![GitHub stars](https://img.shields.io/github/stars/mediconesystems/livewire-datatables?style=flat)](https://github.com/mediconesystems/livewire-datatables/stargazers) - Package to create tables with sorting, filtering, searching, exporting etc.
- [Livewire PowerGrid](https://github.com/Power-Components/livewire-powergrid) [![GitHub stars](https://img.shields.io/github/stars/Power-Components/livewire-powergrid?style=flat)](https://github.com/Power-Components/livewire-powergrid/stargazers) - ⚡ PowerGrid ⚡ creates modern, powerful and easy to customize Datatables.
- [MarkdownX](https://devdojo.com/markdownx) - An advanced Markdown Editor with dropdowns, image uploads, code blocks, and more.
