- [TrackMate](https://github.com/fiji/TrackMate) [![GitHub stars](https://img.shields.io/github/stars/fiji/TrackMate?style=flat)](https://github.com/fiji/TrackMate/stargazers) - User-friendly interface that allows for performing tracking, data visualization, editing results and track analysis in a convenient way.
- [TrackMateR](https://github.com/quantixed/TrackMateR) [![GitHub stars](https://img.shields.io/github/stars/quantixed/TrackMateR?style=flat)](https://github.com/quantixed/TrackMateR/stargazers) - R package to analyze cell migration and particle tracking experiments using outputs from TrackMate.
- [Trackpy](https://soft-matter.github.io/trackpy) - Fast and Flexible Particle-Tracking Toolkit.
- [TracX](https://gitlab.com/csb.ethz/tracx) - MATLAB generic toolbox for cell tracking from various microscopy image modalities such as Bright-field (BF), phase contrast (PhC) or fluorescence (FL) with an automated track quality assessment in
absence of a ground truth.
- [TraJClassifier](https://imagej.net/plugins/trajclassifier) - Fiji plugin that loads trajectories from TrackMate, characterizes them using TraJ and classifiies them into normal diffusion, subdiffusion, confined diffusion and directed/active motion by a random forest approach (through Renjin).
- [QuimP](https://github.com/CellDynamics/QuimP) [![GitHub stars](https://img.shields.io/github/stars/CellDynamics/QuimP?style=flat)](https://github.com/CellDynamics/QuimP/stargazers) - Software for tracking cellular shape changes and dynamic distributions of fluorescent reporters at the cell membrane.
- [Ultrack](https://github.com/royerlab/ultrack) [![GitHub stars](https://img.shields.io/github/stars/royerlab/ultrack?style=flat)](https://github.com/royerlab/ultrack/stargazers) - Versatile cell tracking method for 2D, 3D, and multichannel timelapses, overcoming segmentation challenges in complex tissues.
- [Usiigaci](https://github.com/oist/usiigaci) [![GitHub stars](https://img.shields.io/github/stars/oist/usiigaci?style=flat)](https://github.com/oist/usiigaci/stargazers) - Stain-free cell tracking in phase contrast microscopy enabled by supervised machine learning.
