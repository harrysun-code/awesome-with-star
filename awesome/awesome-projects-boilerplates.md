- [Meteor boilerplate](https://github.com/Differential/meteor-boilerplate) [![GitHub stars](https://img.shields.io/github/stars/Differential/meteor-boilerplate?style=flat)](https://github.com/Differential/meteor-boilerplate/stargazers) Boilerplate for meteor framework.
- [Web Starter Kit](https://github.com/google/web-starter-kit) [![GitHub stars](https://img.shields.io/github/stars/google/web-starter-kit?style=flat)](https://github.com/google/web-starter-kit/stargazers) Web starter kit by Google.
- [Open Designs](http://www.opendesigns.org/) Open Source WebDesign Templates
- [Open Web Design](http://www.openwebdesign.org/) Community of designers and site owners sharing free web design templates
- [OSWD](http://www.oswd.org/) Platform for sharing standards-compliant free web design templates
- [Jekyll Starter Kit](https://github.com/nirgn975/generator-jekyll-starter-kit) [![GitHub stars](https://img.shields.io/github/stars/nirgn975/generator-jekyll-starter-kit?style=flat)](https://github.com/nirgn975/generator-jekyll-starter-kit/stargazers) Jekyll Progressive Web App generator boilerplate.
- [HTML5 UP!](https://html5up.net/) Responsive HTML5 and CSS3 Site Templates.
- [Gulp front](https://zoxon.github.io/gulp-front/) Frontend boilerplate and framework based on gulp, pug, stylus and babel
- [HTML Website Template](https://github.com/fulldecent/html-website-template) [![GitHub stars](https://img.shields.io/github/stars/fulldecent/html-website-template?style=flat)](https://github.com/fulldecent/html-website-template/stargazers) Starting point for collaboratively-edited HTML websites
- [Nextjs Blog Starter](https://creativedesignsguru.com/demo/Nextjs-Blog-Boilerplate/) Starter code for your blog based on Next.js framework

*Boilerplates and template files*
