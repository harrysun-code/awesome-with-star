> A 🎩 list of network layer resources written pure JS.

## High level

> Based, at least, in a CLI interface.

* [airpaste](https://github.com/mafintosh/airpaste) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/airpaste?style=flat)](https://github.com/mafintosh/airpaste/stargazers) – 1-1 network pipe that auto discovers other peers using mdns.
* [blecat](https://github.com/mafintosh/blecat) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/blecat?style=flat)](https://github.com/mafintosh/blecat/stargazers) – 1-1 pipe over bluetooth low energy.
* [deejay](https://github.com/mafintosh/deejay) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/deejay?style=flat)](https://github.com/mafintosh/deejay/stargazers) – Music player that broadcasts to everyone on the same network.
* [dhtkv](https://github.com/maxogden/dhtkv) [![GitHub stars](https://img.shields.io/github/stars/maxogden/dhtkv?style=flat)](https://github.com/maxogden/dhtkv/stargazers) – CLI for storing arbitrary key/value data in the bittorrent mainline DHT.
* [gun](https://github.com/amark/gun) [![GitHub stars](https://img.shields.io/github/stars/amark/gun?style=flat)](https://github.com/amark/gun/stargazers) – A realtime, decentralized, offline-first, graph database engine.
* [hyperpipe](https://github.com/mafintosh/hyperpipe) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/hyperpipe?style=flat)](https://github.com/mafintosh/hyperpipe/stargazers) – Distributed input/output pipe.
* [hypervision](https://github.com/mafintosh/hypervision) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/hypervision?style=flat)](https://github.com/mafintosh/hypervision/stargazers) – P2P Television.
* [instant.io](https://github.com/webtorrent/instant.io) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/instant.io?style=flat)](https://github.com/webtorrent/instant.io/stargazers) – Streaming file transfer over WebTorrent.
* [ipp-printer](https://github.com/watson/ipp-printer) [![GitHub stars](https://img.shields.io/github/stars/watson/ipp-printer?style=flat)](https://github.com/watson/ipp-printer/stargazers) – Create a printer on your network.
* [peercast](https://github.com/mafintosh/peercast) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/peercast?style=flat)](https://github.com/mafintosh/peercast/stargazers) – Like peerflix but for Chromecast.
* [peerflix](https://github.com/mafintosh/peerflix) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/peerflix?style=flat)](https://github.com/mafintosh/peerflix/stargazers) – Streaming torrent client.
* [peervisionary](https://github.com/mafintosh/peervisionary) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/peervisionary?style=flat)](https://github.com/mafintosh/peervisionary/stargazers) – Stream p2p content over your local network.
* [peerwiki](https://github.com/mafintosh/peerwiki) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/peerwiki?style=flat)](https://github.com/mafintosh/peerwiki/stargazers) – browse all of wikipedia using bittorrent.
* [screencat](https://github.com/maxogden/screencat) [![GitHub stars](https://img.shields.io/github/stars/maxogden/screencat?style=flat)](https://github.com/maxogden/screencat/stargazers) – WebRTC screensharing app.
* [signalhub](https://github.com/mafintosh/signalhub) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/signalhub?style=flat)](https://github.com/mafintosh/signalhub/stargazers) – Simple signalling server that can be used to coordinate handshaking with webrtc or other fun stuff.
* [torrent-mount](https://github.com/mafintosh/torrent-mount) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/torrent-mount?style=flat)](https://github.com/mafintosh/torrent-mount/stargazers) – Mount a torrent (or magnet link) as a filesystem in real time using torrent-stream and fuse.
* [webcat](https://github.com/mafintosh/webcat) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/webcat?style=flat)](https://github.com/mafintosh/webcat/stargazers) – pipe across the web using WebRTC.
* [websocketd](https://github.com/joewalnes/websocketd) [![GitHub stars](https://img.shields.io/github/stars/joewalnes/websocketd?style=flat)](https://github.com/joewalnes/websocketd/stargazers) – Turn any program that uses stdin/stdout into a WebSocket server.
* [webtorrent](https://github.com/webtorrent/webtorrent) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/webtorrent?style=flat)](https://github.com/webtorrent/webtorrent/stargazers) – BitTorrent over WebRTC.
* [wifi-triangulate](https://github.com/watson/wifi-triangulate) [![GitHub stars](https://img.shields.io/github/stars/watson/wifi-triangulate?style=flat)](https://github.com/watson/wifi-triangulate/stargazers) – Finds your current position on planet earth using the wifi access point.

## Modules

> Do one thing well.

* [airplay-server](https://github.com/watson/airplay-server) [![GitHub stars](https://img.shields.io/github/stars/watson/airplay-server?style=flat)](https://github.com/watson/airplay-server/stargazers) – A low level AirPlay server.
* [castnow](https://github.com/xat/chromecast-player) [![GitHub stars](https://img.shields.io/github/stars/xat/chromecast-player?style=flat)](https://github.com/xat/chromecast-player/stargazers) – simple chromecast player.
* [discovery-swarm](https://github.com/mafintosh/discovery-swarm) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/discovery-swarm?style=flat)](https://github.com/mafintosh/discovery-swarm/stargazers) – A network swarm that uses [discovery-channel](https://github.com/maxogden/discovery-channel) [![GitHub stars](https://img.shields.io/github/stars/maxogden/discovery-channel?style=flat)](https://github.com/maxogden/discovery-channel/stargazers) to find peers. Also check [webrtc-swarm](https://github.com/mafintosh/webrtc-swarm) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/webrtc-swarm?style=flat)](https://github.com/mafintosh/webrtc-swarm/stargazers).
* [dns-packet](https://github.com/mafintosh/dns-packet) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/dns-packet?style=flat)](https://github.com/mafintosh/dns-packet/stargazers) – Abstract-encoding compliant module for encoding / decoding DNS packets. Also see [dns-socket](https://github.com/mafintosh/dns-socket) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/dns-socket?style=flat)](https://github.com/mafintosh/dns-socket/stargazers).
* [etcdjs](https://github.com/mafintosh/etcdjs) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/etcdjs?style=flat)](https://github.com/mafintosh/etcdjs/stargazers) – Low level etcd v2 client written in Javascript with failover support.
* [geocode-wifi](https://github.com/watson/geocode-wifi) [![GitHub stars](https://img.shields.io/github/stars/watson/geocode-wifi?style=flat)](https://github.com/watson/geocode-wifi/stargazers) – Get yours latitude/longitude based on your wifi access point.
* [hash-to-port](https://github.com/mafintosh/hash-to-port) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/hash-to-port?style=flat)](https://github.com/mafintosh/hash-to-port/stargazers) – Hash a value to a valid port.
* [hyperdb](https://github.com/mafintosh/hyperdb) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/hyperdb?style=flat)](https://github.com/mafintosh/hyperdb/stargazers) – Distributed scalable database.
* [ip-packet](https://github.com/mafintosh/ip-packet) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/ip-packet?style=flat)](https://github.com/mafintosh/ip-packet/stargazers) – Encode/decode raw ip packets.
* [magnet-uri](https://github.com/webtorrent/magnet-uri) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/magnet-uri?style=flat)](https://github.com/webtorrent/magnet-uri/stargazers) – Parse a magnet URI and return an object of keys/values.
* [mutex-server](https://github.com/samchon/mutex-server) [![GitHub stars](https://img.shields.io/github/stars/samchon/mutex-server?style=flat)](https://github.com/samchon/mutex-server/stargazers) - Critical section controllers, like mutex and semaphore, in the network level.

* [network-simulator](https://github.com/substack/network-simulator) [![GitHub stars](https://img.shields.io/github/stars/substack/network-simulator?style=flat)](https://github.com/substack/network-simulator/stargazers) – Simulate a low-level computer network.
* [rtsp-stream](https://github.com/watson/rtsp-stream) [![GitHub stars](https://img.shields.io/github/stars/watson/rtsp-stream?style=flat)](https://github.com/watson/rtsp-stream/stargazers) - A transport agnostic RTSP serial multiplexer module for Node.

* [torrent-discovery](https://github.com/webtorrent/torrent-discovery) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/torrent-discovery?style=flat)](https://github.com/webtorrent/torrent-discovery/stargazers) – Discover BitTorrent and WebTorrent peers.

## Protocols

> Implementation of protocols specs in pure javascript.

* [airswarm](https://github.com/mafintosh/airswarm) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/airswarm?style=flat)](https://github.com/mafintosh/airswarm/stargazers) – Network swarm that automagically discovers other peers on the network using multicast dns.
* [bittorrent-dht](https://github.com/webtorrent/bittorrent-dht) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/bittorrent-dht?style=flat)](https://github.com/webtorrent/bittorrent-dht/stargazers) – BitTorrent DHT protocol implementation.
* [bittorrent-protocol](https://github.com/webtorrent/bittorrent-protocol) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/bittorrent-protocol?style=flat)](https://github.com/webtorrent/bittorrent-protocol/stargazers) – BitTorrent peer wire protocol implementation.
* [bittorrent-tracker](https://github.com/webtorrent/bittorrent-tracker) [![GitHub stars](https://img.shields.io/github/stars/webtorrent/bittorrent-tracker?style=flat)](https://github.com/webtorrent/bittorrent-tracker/stargazers) – BitTorrent tracker (client & server) implementation
* [bonjour](https://github.com/watson/bonjour) [![GitHub stars](https://img.shields.io/github/stars/watson/bonjour?style=flat)](https://github.com/watson/bonjour/stargazers) – A Bonjour/Zeroconf protocol implementation.
* [castv2](https://github.com/thibauts/node-castv2) [![GitHub stars](https://img.shields.io/github/stars/thibauts/node-castv2?style=flat)](https://github.com/thibauts/node-castv2/stargazers) – An implementation of the Chromecast CASTV2 protocol.
* [dht-rpc](https://github.com/mafintosh/dht-rpc) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/dht-rpc?style=flat)](https://github.com/mafintosh/dht-rpc/stargazers) – Make RPC calls over a [Kademlia](https://pdos.csail.mit.edu/~petar/papers/maymounkov-kademlia-lncs.pdf) based DHT.
* [dns-discovery](https://github.com/mafintosh/dns-discovery) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/dns-discovery?style=flat)](https://github.com/mafintosh/dns-discovery/stargazers) – Discovery peers in a distributed system using regular dns and multicast dns.
* [hypercore](https://github.com/mafintosh/hypercore) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/hypercore?style=flat)](https://github.com/mafintosh/hypercore/stargazers) – A p2p network for distributing and replicating static feeds of binary data.
* [ipfs](https://github.com/ipfs/js-ipfs-api) [![GitHub stars](https://img.shields.io/github/stars/ipfs/js-ipfs-api?style=flat)](https://github.com/ipfs/js-ipfs-api/stargazers) – The InterPlanetary File System, a new peer-to-peer hypermedia protocol.
* [k-bucket](https://github.com/tristanls/k-bucket) [![GitHub stars](https://img.shields.io/github/stars/tristanls/k-bucket?style=flat)](https://github.com/tristanls/k-bucket/stargazers) – Kademlia DHT K-bucket implementation as a binary tree.
* [k-rpc](https://github.com/mafintosh/k-rpc) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/k-rpc?style=flat)](https://github.com/mafintosh/k-rpc/stargazers) – Implementation of the k-rpc protocol used the BitTorrent DHT. Also see [k-rpc-socket](https://github.com/mafintosh/k-rpc-socket) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/k-rpc-socket?style=flat)](https://github.com/mafintosh/k-rpc-socket/stargazers).
* [mdns](https://github.com/agnat/node_mdns) [![GitHub stars](https://img.shields.io/github/stars/agnat/node_mdns?style=flat)](https://github.com/agnat/node_mdns/stargazers) – mdns/zeroconf/bonjour service discovery.
* [multicast-dns](https://github.com/mafintosh/multicast-dns) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/multicast-dns?style=flat)](https://github.com/mafintosh/multicast-dns/stargazers) – Low level multicast-dns implementation.
* [peervision](https://github.com/mafintosh/peervision) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/peervision?style=flat)](https://github.com/mafintosh/peervision/stargazers) – Live p2p streaming protocol.
* [polo](https://github.com/mafintosh/polo) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/polo?style=flat)](https://github.com/mafintosh/polo/stargazers) – A zero configuration service discovery module.
* [rtsp-server](https://github.com/watson/rtsp-server) [![GitHub stars](https://img.shields.io/github/stars/watson/rtsp-server?style=flat)](https://github.com/watson/rtsp-server/stargazers) – A low level module for creating RTSP servers.
* [utp-native](https://github.com/mafintosh/utp-native) [![GitHub stars](https://img.shields.io/github/stars/mafintosh/utp-native?style=flat)](https://github.com/mafintosh/utp-native/stargazers) – micro transport protocol, a network protocol similar to tcp that runs on top of udp.
