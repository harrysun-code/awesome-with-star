- [wasm-pack-action](https://github.com/jetli/wasm-pack-action) [![GitHub stars](https://img.shields.io/github/stars/jetli/wasm-pack-action?style=flat)](https://github.com/jetli/wasm-pack-action/stargazers) - Github action to install `wasm-pack` by downloading the executable to speed up CI/CD.
- [wasm-bindgen-action](https://github.com/jetli/wasm-bindgen-action) [![GitHub stars](https://img.shields.io/github/stars/jetli/wasm-bindgen-action?style=flat)](https://github.com/jetli/wasm-bindgen-action/stargazers) - Github action to install `wasm-bindgen` by downloading the executable to speed up CI/CD.
- [cargo-web](https://github.com/koute/cargo-web) [![GitHub stars](https://img.shields.io/github/stars/koute/cargo-web?style=flat)](https://github.com/koute/cargo-web/stargazers) - A Cargo subcommand for the client-side Web.
- [Trunk](https://github.com/thedodd/trunk) [![GitHub stars](https://img.shields.io/github/stars/thedodd/trunk?style=flat)](https://github.com/thedodd/trunk/stargazers) - Build, bundle & ship your Rust Wasm application to the web.
- [trunk-action](https://github.com/jetli/trunk-action) [![GitHub stars](https://img.shields.io/github/stars/jetli/trunk-action?style=flat)](https://github.com/jetli/trunk-action/stargazers) - Github action to install `Trunk` by downloading the executable to speed up CI/CD.
- [wabt](https://github.com/WebAssembly/wabt) [![GitHub stars](https://img.shields.io/github/stars/WebAssembly/wabt?style=flat)](https://github.com/WebAssembly/wabt/stargazers) - The WebAssembly Binary Toolkit, for the `wasm-strip` and `wasm-objdump` tools to reduce .wasm file size.
- [binaryen](https://github.com/WebAssembly/binaryen) [![GitHub stars](https://img.shields.io/github/stars/WebAssembly/binaryen?style=flat)](https://github.com/WebAssembly/binaryen/stargazers) - Compiler infrastructure and toolchain library for WebAssembly, for the `wasm-opt` tool to reduce .wasm file size.
- [Tauri](https://github.com/tauri-apps/tauri) [![GitHub stars](https://img.shields.io/github/stars/tauri-apps/tauri?style=flat)](https://github.com/tauri-apps/tauri/stargazers) - Tauri is a framework for building tiny, blazingly fast binaries for all major desktop platforms. Developers can integrate any front-end framework that compiles to HTML, JS and CSS for building their user interface. The backend of the application is a rust-sourced binary with an API that the front-end can interact with.
- [yew-fmt](https://github.com/schvv31n/yew-fmt) [![GitHub stars](https://img.shields.io/github/stars/schvv31n/yew-fmt?style=flat)](https://github.com/schvv31n/yew-fmt/stargazers) - A configurable extension to `rustfmt` for formatting Yew HTML.
