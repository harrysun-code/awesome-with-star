> QR code (abbreviated from Quick Response code) is a type of matrix barcode (or two-dimensional barcode) first designed in 1994 for the automotive industry in Japan.
>
> *Wikipedia* ~ https://en.wikipedia.org/wiki/QR_code

## Apps

### Readers

- [Web App](https://github.com/gokulkrishh/qrcodescan.in) [![GitHub stars](https://img.shields.io/github/stars/gokulkrishh/qrcodescan.in?style=flat)](https://github.com/gokulkrishh/qrcodescan.in/stargazers) - A progressive web application to scan QR codes.

### File Transfer

- [qrcp](https://github.com/claudiodangelis/qrcp) [![GitHub stars](https://img.shields.io/github/stars/claudiodangelis/qrcp?style=flat)](https://github.com/claudiodangelis/qrcp/stargazers) - Transfer files over Wi-Fi from your computer to a mobile device by scanning a QR code without leaving the terminal.
- [qr-filetransfer](https://github.com/sdushantha/qr-filetransfer) [![GitHub stars](https://img.shields.io/github/stars/sdushantha/qr-filetransfer?style=flat)](https://github.com/sdushantha/qr-filetransfer/stargazers) - Transfer files over Wi-Fi between your computer and your smartphone from the terminal.

## CLI

- [qrencode](https://fukuchi.org/works/qrencode) - Command line tool to generate QR codes.
- [qrap](https://github.com/make-github-pseudonymous-again/qrap) [![GitHub stars](https://img.shields.io/github/stars/make-github-pseudonymous-again/qrap?style=flat)](https://github.com/make-github-pseudonymous-again/qrap/stargazers) - Generate a QR code payload for Wi-Fi access point credentials.
- [spqr](https://github.com/make-github-pseudonymous-again/spqr) [![GitHub stars](https://img.shields.io/github/stars/make-github-pseudonymous-again/spqr?style=flat)](https://github.com/make-github-pseudonymous-again/spqr/stargazers) - Generate a QR code payload for initiating a SEPA transfer.
- [qrscan](https://github.com/sayanarijit/qrscan) [![GitHub stars](https://img.shields.io/github/stars/sayanarijit/qrscan?style=flat)](https://github.com/sayanarijit/qrscan/stargazers) - Scan a QR code in the terminal using the system camera or a given image.
- [qrcode.show](https://qrcode.show) - Generate QR codes using curl.

## Libraries

In this section `R` means the library allows to read QR codes and `W` means
that the library allows to generate QR codes.

### C

- [libqrencode `W`](https://github.com/fukuchi/libqrencode) [![GitHub stars](https://img.shields.io/github/stars/fukuchi/libqrencode?style=flat)](https://github.com/fukuchi/libqrencode/stargazers) - A fast and compact library for encoding data in a QR code symbol.

### C, C++, JavaScript, Python, Java, Rust

- [QR-Code-generator `W`](https://github.com/nayuki/QR-Code-generator) [![GitHub stars](https://img.shields.io/github/stars/nayuki/QR-Code-generator?style=flat)](https://github.com/nayuki/QR-Code-generator/stargazers) - High-quality QR code generator library in Java, TypeScript/JavaScript, Python, C++, C, Rust.

### Java

- [Qart `W`](https://github.com/scola/Qart) [![GitHub stars](https://img.shields.io/github/stars/scola/Qart?style=flat)](https://github.com/scola/Qart/stargazers) - An app to merge a picture and a QR code.

### JavaScript

- [instascan `R`](https://github.com/schmich/instascan) [![GitHub stars](https://img.shields.io/github/stars/schmich/instascan?style=flat)](https://github.com/schmich/instascan/stargazers) - HTML5 QR code scanner using your webcam.
- [qrious `W`](https://github.com/neocotic/qrious) [![GitHub stars](https://img.shields.io/github/stars/neocotic/qrious?style=flat)](https://github.com/neocotic/qrious/stargazers) - Pure JavaScript library for QR code generation using canvas.

### TypeScript

- [jsQR `R`](https://github.com/cozmo/jsQR) [![GitHub stars](https://img.shields.io/github/stars/cozmo/jsQR?style=flat)](https://github.com/cozmo/jsQR/stargazers) - A pure JavaScript QR code reading library.

### React

- [qrcode.react `W`](https://github.com/zpao/qrcode.react) [![GitHub stars](https://img.shields.io/github/stars/zpao/qrcode.react?style=flat)](https://github.com/zpao/qrcode.react/stargazers) - A `<QRCode/>` component for use with React.

### React Native

- [react-native-qrcode-scanner `R`](https://github.com/moaazsidat/react-native-qrcode-scanner) [![GitHub stars](https://img.shields.io/github/stars/moaazsidat/react-native-qrcode-scanner?style=flat)](https://github.com/moaazsidat/react-native-qrcode-scanner/stargazers) - A QR code scanner component for React Native.

### Vue

- [vue-qrcode-reader `R`](https://github.com/gruhn/vue-qrcode-reader) [![GitHub stars](https://img.shields.io/github/stars/gruhn/vue-qrcode-reader?style=flat)](https://github.com/gruhn/vue-qrcode-reader/stargazers) - A set of Vue.js components for detecting and decoding QR codes.

### Python

- [sylnsfar/qrcode `W`](https://github.com/sylnsfar/qrcode) [![GitHub stars](https://img.shields.io/github/stars/sylnsfar/qrcode?style=flat)](https://github.com/sylnsfar/qrcode/stargazers) - Artistic QR code in Python (can produce GIFs).

### Ruby
- [rqrcode `W`](https://github.com/whomwah/rqrcode) [![GitHub stars](https://img.shields.io/github/stars/whomwah/rqrcode?style=flat)](https://github.com/whomwah/rqrcode/stargazers) - A neat Ruby QR code generator exporting to `svg`, `png` and `ansi` with configurable size, color and other attributes.

### Objective-C

- [SGQRCode `R`](https://github.com/kingsic/SGQRCode) [![GitHub stars](https://img.shields.io/github/stars/kingsic/SGQRCode?style=flat)](https://github.com/kingsic/SGQRCode/stargazers) - Easy to use QR code scan library for iOS.

### Swift

- [EFQRCode `W`](https://github.com/EFPrefix/EFQRCode) [![GitHub stars](https://img.shields.io/github/stars/EFPrefix/EFQRCode?style=flat)](https://github.com/EFPrefix/EFQRCode/stargazers) - A better way to operate QR code in Swift, support iOS, macOS, watchOS and tvOS.

### C\#

- [QRCoder `W`](https://github.com/codebude/QRCoder) [![GitHub stars](https://img.shields.io/github/stars/codebude/QRCoder?style=flat)](https://github.com/codebude/QRCoder/stargazers) - A pure C# Open Source QR code implementation.

### Android

- [BGAQRCode-Android `RW`](https://github.com/bingoogolapple/BGAQRCode-Android) [![GitHub stars](https://img.shields.io/github/stars/bingoogolapple/BGAQRCode-Android?style=flat)](https://github.com/bingoogolapple/BGAQRCode-Android/stargazers) - QR code reader and generator written in C.
- [AwesomeQRCode (Kotlin) `W`](https://github.com/SumiMakito/AwesomeQRCode) [![GitHub stars](https://img.shields.io/github/stars/SumiMakito/AwesomeQRCode?style=flat)](https://github.com/SumiMakito/AwesomeQRCode/stargazers) - An awesome QR code generator for Android (can produce GIFs).

### PHP

- [php-qrcode-detector-decoder `R`](https://github.com/khanamiryan/php-qrcode-detector-decoder) [![GitHub stars](https://img.shields.io/github/stars/khanamiryan/php-qrcode-detector-decoder?style=flat)](https://github.com/khanamiryan/php-qrcode-detector-decoder/stargazers) - PHP library to detect and decode QR codes.
- [simple-qrcode `W`](https://github.com/SimpleSoftwareIO/simple-qrcode) [![GitHub stars](https://img.shields.io/github/stars/SimpleSoftwareIO/simple-qrcode?style=flat)](https://github.com/SimpleSoftwareIO/simple-qrcode/stargazers) - An easy-to-use PHP QR code generator.
- [endroid/qr-code `W`](https://github.com/endroid/qr-code) [![GitHub stars](https://img.shields.io/github/stars/endroid/qr-code?style=flat)](https://github.com/endroid/qr-code/stargazers) - QR code generator.
- [BaconQRCode `W`](https://github.com/Bacon/BaconQRCode) [![GitHub stars](https://img.shields.io/github/stars/Bacon/BaconQRCode?style=flat)](https://github.com/Bacon/BaconQRCode/stargazers) - QR code generator for PHP.
