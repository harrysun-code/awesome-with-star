## Bindings

* [RxAndroid](https://github.com/ReactiveX/RxAndroid) [![GitHub stars](https://img.shields.io/github/stars/ReactiveX/RxAndroid?style=flat)](https://github.com/ReactiveX/RxAndroid/stargazers) - Android specific bindings for RxJava.
* [RxBinding](https://github.com/JakeWharton/RxBinding) [![GitHub stars](https://img.shields.io/github/stars/JakeWharton/RxBinding?style=flat)](https://github.com/JakeWharton/RxBinding/stargazers) - RxJava binding APIs for Android UI widgets from the platform and support libraries.
* [rx-preferences](https://github.com/f2prateek/rx-preferences) [![GitHub stars](https://img.shields.io/github/stars/f2prateek/rx-preferences?style=flat)](https://github.com/f2prateek/rx-preferences/stargazers) - Reactive `SharedPreferences` for Android.
* [RxPermissions](https://github.com/tbruyelle/RxPermissions) [![GitHub stars](https://img.shields.io/github/stars/tbruyelle/RxPermissions?style=flat)](https://github.com/tbruyelle/RxPermissions/stargazers) - Android M runtime permissions powered by RxJava.
* [SQLBrite](https://github.com/square/sqlbrite) [![GitHub stars](https://img.shields.io/github/stars/square/sqlbrite?style=flat)](https://github.com/square/sqlbrite/stargazers) - A lightweight wrapper around SQLiteOpenHelper and ContentResolver which introduces reactive stream semantics to queries.
* [Android-ReactiveLocation](https://github.com/mcharmas/Android-ReactiveLocation) [![GitHub stars](https://img.shields.io/github/stars/mcharmas/Android-ReactiveLocation?style=flat)](https://github.com/mcharmas/Android-ReactiveLocation/stargazers) - Small library that wraps Google Play Service API in brilliant RxJava Observables reducing boilerplate to minimum.
* [ReactiveNetwork](https://github.com/pwittchen/ReactiveNetwork) [![GitHub stars](https://img.shields.io/github/stars/pwittchen/ReactiveNetwork?style=flat)](https://github.com/pwittchen/ReactiveNetwork/stargazers) - Android library listening network connection state and change of the WiFi signal strength with RxJava Observables.
* [ReactiveSensors](https://github.com/pwittchen/ReactiveSensors) [![GitHub stars](https://img.shields.io/github/stars/pwittchen/ReactiveSensors?style=flat)](https://github.com/pwittchen/ReactiveSensors/stargazers) - Android library monitoring hardware sensors with RxJava Observables.
* [RxPalette](https://github.com/hzsweers/RxPalette) [![GitHub stars](https://img.shields.io/github/stars/hzsweers/RxPalette?style=flat)](https://github.com/hzsweers/RxPalette/stargazers) - RxJava bindings for the Palette library on Android.
* [rxjava-jdbc](https://github.com/davidmoten/rxjava-jdbc) [![GitHub stars](https://img.shields.io/github/stars/davidmoten/rxjava-jdbc?style=flat)](https://github.com/davidmoten/rxjava-jdbc/stargazers) - Efficient execution and functional composition of database calls using jdbc and RxJava Observables.
* [rxjava-file](https://github.com/davidmoten/rxjava-file) [![GitHub stars](https://img.shields.io/github/stars/davidmoten/rxjava-file?style=flat)](https://github.com/davidmoten/rxjava-file/stargazers) - RxJava observables for files including NIO events.
* [RxTuples](https://github.com/pakoito/RxTuples) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxTuples?style=flat)](https://github.com/pakoito/RxTuples/stargazers) - Simple tuples to use with RxJava.
* [RxAnimationBinding](https://github.com/blipinsk/RxAnimationBinding) [![GitHub stars](https://img.shields.io/github/stars/blipinsk/RxAnimationBinding?style=flat)](https://github.com/blipinsk/RxAnimationBinding/stargazers) - RxJava binding APIs for Android's animations.

String and Byte operators for RxJava.
* [RxJavaComputationExpressions](https://github.com/ReactiveX/RxJavaComputationExpressions) [![GitHub stars](https://img.shields.io/github/stars/ReactiveX/RxJavaComputationExpressions?style=flat)](https://github.com/ReactiveX/RxJavaComputationExpressions/stargazers) - Computation expressions for RxJava.
* [rxjava-extras](https://github.com/davidmoten/rxjava-extras) [![GitHub stars](https://img.shields.io/github/stars/davidmoten/rxjava-extras?style=flat)](https://github.com/davidmoten/rxjava-extras/stargazers) - Utilities for use with RxJava.
* [RxActions](https://github.com/pakoito/RxActions) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxActions?style=flat)](https://github.com/pakoito/RxActions/stargazers) - Simple ActionN composition to use with RxJava.
* [RxRelay](https://github.com/JakeWharton/RxRelay) [![GitHub stars](https://img.shields.io/github/stars/JakeWharton/RxRelay?style=flat)](https://github.com/JakeWharton/RxRelay/stargazers) - RxJava types that are both an Observable and an Action1.
* [Frodo](https://github.com/android10/frodo) [![GitHub stars](https://img.shields.io/github/stars/android10/frodo?style=flat)](https://github.com/android10/frodo/stargazers) - Android Library for Logging RxJava Observables and Subscribers.
* [RxPartialApplication](https://github.com/pakoito/RxPartialApplication) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxPartialApplication?style=flat)](https://github.com/pakoito/RxPartialApplication/stargazers) - Simple partial application for FuncN and ActionN on RxJava.
* [RxCurrying](https://github.com/pakoito/RxCurrying) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxCurrying?style=flat)](https://github.com/pakoito/RxCurrying/stargazers) - Simple currying for FuncN and ActionN on RxJava.
* [RxEither](https://github.com/eleventigers/rxeither) [![GitHub stars](https://img.shields.io/github/stars/eleventigers/rxeither?style=flat)](https://github.com/eleventigers/rxeither/stargazers) - Either type for RxJava.
* [RxReplayingShare](https://github.com/JakeWharton/RxReplayingShare) [![GitHub stars](https://img.shields.io/github/stars/JakeWharton/RxReplayingShare?style=flat)](https://github.com/JakeWharton/RxReplayingShare/stargazers) - An RxJava transformer which combines replay(1), publish(), and refCount() operators.
* [RxFunctions](https://github.com/pakoito/RxFunctions) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxFunctions?style=flat)](https://github.com/pakoito/RxFunctions/stargazers) - Advanced Function composition to use with RxJava.
* [rxlint](https://bitbucket.org/littlerobots/rxlint) - An Android lint rule for RxJava code.
* [RxComprehensions](https://github.com/pakoito/RxComprehensions) [![GitHub stars](https://img.shields.io/github/stars/pakoito/RxComprehensions?style=flat)](https://github.com/pakoito/RxComprehensions/stargazers) - Reduce boilerplate in RxJava by abstracting chained flatMaps, concatMaps and switchMaps.

## Articles

* [Rx glitches aren't actually a problem](http://staltz.com/rx-glitches-arent-actually-a-problem.html) - Glitches are temporary inconsistencies emitted by Observables. André Staltz looks at why it's not really a problem.
* [RxJava's repeatWhen and retryWhen, explained](http://blog.danlew.net/2016/01/25/rxjavas-repeatwhen-and-retrywhen-explained/) - `repeatWhen` and `retryWhen` are fairly baffling at first glance. Dan Lew explains the operators in depth.
* [RxJava - The Problem with Subjects](http://tomstechnicalblog.blogspot.co.uk/2016/03/rxjava-problem-with-subjects.html) - Thomas Nield explains why `Subject` is not a panacea.
