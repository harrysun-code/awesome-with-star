## Extensions

*Extensions add to the functionality of Vorpal - programmatically or live.*

#### POSIX implementations

- [less](https://github.com/vorpaljs/vorpal-less) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-less?style=flat)](https://github.com/vorpaljs/vorpal-less/stargazers) - Implementation of the `less` command.
- [grep](https://github.com/vorpaljs/vorpal-grep) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-grep?style=flat)](https://github.com/vorpaljs/vorpal-grep/stargazers) - Implementation of the `grep` command.

#### Development tools

- [repl](https://github.com/vorpaljs/vorpal-repl) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-repl?style=flat)](https://github.com/vorpaljs/vorpal-repl/stargazers) - Drops your CLI into a REPL within your app's context.
- [tour](https://github.com/vorpaljs/vorpal-tour) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-tour?style=flat)](https://github.com/vorpaljs/vorpal-tour/stargazers) - Build an interactive tour for your Vorpal app.
- [watch](https://github.com/vantagejs/vantage-watch) [![GitHub stars](https://img.shields.io/github/stars/vantagejs/vantage-watch?style=flat)](https://github.com/vantagejs/vantage-watch/stargazers) - Updates your live Vorpal extensions in realtime.
- [log](https://github.com/AljoschaMeyer/vorpal-log) [![GitHub stars](https://img.shields.io/github/stars/AljoschaMeyer/vorpal-log?style=flat)](https://github.com/AljoschaMeyer/vorpal-log/stargazers) - Advanced logging utility.
- [set-or-print](https://github.com/AljoschaMeyer/vorpal-setorprint) [![GitHub stars](https://img.shields.io/github/stars/AljoschaMeyer/vorpal-setorprint?style=flat)](https://github.com/AljoschaMeyer/vorpal-setorprint/stargazers) - Easily set / get arguments with commands.

#### Realtime

- [use](https://github.com/vorpaljs/vorpal-use) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-use?style=flat)](https://github.com/vorpaljs/vorpal-use/stargazers) - Import Vorpal extensions live: while your app is running.

#### Fun

- [hacker-news](https://github.com/vorpaljs/vorpal-hacker-news) [![GitHub stars](https://img.shields.io/github/stars/vorpaljs/vorpal-hacker-news?style=flat)](https://github.com/vorpaljs/vorpal-hacker-news/stargazers) - Pulls the top trending items from Hacker News.

#### Toolbox
- [rc](https://github.com/subk/vorpal-rc) [![GitHub stars](https://img.shields.io/github/stars/subk/vorpal-rc?style=flat)](https://github.com/subk/vorpal-rc/stargazers) - Simple .rc file support.
- [comment](https://github.com/subk/vorpal-comment) [![GitHub stars](https://img.shields.io/github/stars/subk/vorpal-comment?style=flat)](https://github.com/subk/vorpal-comment/stargazers) - Bash-like comment support.
- [as-default](https://github.com/ialpert/vorpal-as-default) [![GitHub stars](https://img.shields.io/github/stars/ialpert/vorpal-as-default?style=flat)](https://github.com/ialpert/vorpal-as-default/stargazers) - Triggers Vorpal task in case it runs without any specificly defined task.
